PDF Tools Overview

PDF Cutter (Online Version)

This tool allows you to cut a PDF file by selecting specific pages from a range. It can be used online and requires an active internet connection because it loads the pdf-lib and downloadjs libraries from external sources.

Go to PDF Cutter (Online)

PDF Cutter (Offline Version)

This is the offline version of the PDF Cutter tool. It allows you to cut a PDF without requiring an internet connection. The necessary libraries, like pdf-lib and downloadjs, need to be downloaded to your machine and placed in a directory called "js" which should be in the same directory as the HTML file.

Go to PDF Cutter (Offline)

PDF Merge (Online Version)

This tool lets you merge multiple PDF files into a single document. The order of files can be changed by dragging and dropping them. It can be used online and requires an active internet connection because it loads the pdf-lib and Sortable libraries from external sources.

Go to PDF Merge (Online)

PDF Merge (Offline Version)

This is the offline version of the PDF Merge tool. The necessary libraries, like pdf-lib and Sortable, need to be downloaded to your machine and placed in a directory called "js" which should be in the same directory as the HTML file.

Go to PDF Merge (Offline)
